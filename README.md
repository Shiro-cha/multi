# multicast-search

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run client/Ar.ts
```

To compile to Javascript:

```bash
bun compile
```

To compile and run:

```bash
bun cli
```

To run as daemon:

```bash
bun daemon
```

This project was created using `bun init` in bun v1.0.3. [Bun](https://bun.sh) is a fast all-in-one JavaScript runtime.
